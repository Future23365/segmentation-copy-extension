# 文字拆分复制

一个用于快速复制网页局部文本的 Chrome 扩展。

扩展会把选中文本、当前文本块、链接文本、链接地址或手动输入内容拆成多个可点击片段。点击任意片段后，会直接复制该片段，适合处理资源标题、下载页标题、链接文本、URL、文件名等中英文混合内容。

## 功能特性

- 支持通过右键菜单打开拆分面板。
- 支持读取当前网页选中文本。
- 未选中文本时，可尝试读取右键位置附近的当前文本块。
- 右键链接时优先展示链接文本，并可切换到链接地址。
- 支持点击扩展图标，在弹窗中输入或粘贴文本后拆分。
- 支持 URL、中文、英文、数字、符号和常见资源参数的结构化拆分。
- 点击拆分片段即可复制，复制成功后面板会自动收起。

## 安装使用

这是一个无需构建的 Manifest V3 扩展，可以直接以未打包扩展方式加载。

1. 打开 Chrome 扩展管理页：

```text
chrome://extensions/
```

2. 打开右上角的“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择当前项目目录。
5. 加载完成后，可以把扩展图标固定到浏览器工具栏。

## 使用方式

### 右键拆分网页文本

在网页中选中一段文本后右键，点击：

```text
拆分并复制片段
```

扩展会在鼠标附近打开拆分面板。点击面板中的任意片段即可复制。

### 右键当前文本块

如果没有选中文本，也可以直接在标题、链接、列表项、按钮或普通文本块上右键。扩展会尝试读取当前位置附近的文本内容并拆分。

为了避免误读整页内容，当前文本块读取上限为 1000 字。超过上限时，需要手动框选需要拆分的部分。

### 右键链接

右键链接时，扩展会优先拆分链接文本。面板中会显示“切换到链接地址”按钮，可切换为拆分链接 URL。

### 弹窗手动输入

点击浏览器工具栏中的扩展图标，可以打开弹窗。弹窗会优先读取当前页面的选中文本，也可以手动粘贴标题、URL 或文件名进行拆分。

## 分词策略

当前版本采用结构规则拆分，不做复杂 NLP 语义识别。

- URL 会按协议、域名、路径、查询参数和 hash 拆分。
- 中文连续文本会作为完整片段保留，不强行拆成中文词语。
- 英文标题会尽量保留为短语。
- 常见资源参数会作为独立片段，例如 `10-bit`、`1080p`、`HEVC`、`BDRip`、`Fin`。
- 结构符号和标点会独立成片段，例如 `/`、`[`、`]`、`!`、`！`。

```text

├── background.js        # 右键菜单注册与消息分发

├── popup.html           # 扩展弹窗页面

├── src/

│   └── clipboard.js     # 复制封装
└── icons/               # 扩展图标
```

## 开发说明

项目没有构建步骤，修改文件后在 Chrome 扩展管理页点击刷新按钮即可重新加载。

调试建议：

- 调试右键菜单和消息分发时查看扩展的 service worker 控制台。
- 调试网页浮层时查看当前网页的 DevTools。
- 调试弹窗时右键扩展弹窗并选择检查。

## 权限说明

扩展当前使用以下权限：

- `contextMenus`：注册右键菜单。
- `activeTab`：读取当前活动标签页的选中文本。
- `clipboardWrite`：点击片段后写入剪贴板。

内容脚本匹配范围为：

```text
<all_urls>
```

部分浏览器内部页面，例如 `chrome://` 页面，不允许注入内容脚本，因此无法使用网页浮层功能。
